About

Performing updates on routers is quite different from full Linux distribution. It's not sustainable to do release upgrade via a packet manager. Instead it's usually required to re-flash the system image. Depending on the installed packages an image rebuild may be to complex for regular users. A more convenient way is needed.

This project implements an "image as a service" server which provides custom build images depending on installed packages. These updates are installable via a one-click installation in the web interface.

The server side implementation uses established tools like LEDE's ImageBuilder to provide an generic approach for image creating. Eventually all devices of the OpenWrt/LEDE community including several community-mesh firmware projects should profit of this project.

The project started via a GSoC project called "LibreMesh Attended Sysupgrade"

Currently the client side is implemented for LuCi1 while all JavaScript code should be compatible with the upcomming luci-ng. To use this project on a client the new package luci-app-attendedsysupgrade (currently only in LEDE snapshots) is required.

All server related code can be found in the current GitHub repository with additional information in the README
